• HTML Element:

    • Anything in between same html tags is called element.

  • Attribute:

    • An attribute adds specificty to an element. It is placed in opening tag of that element.
    Class,reference,relation etc. are examples of attributes.

  • HTML Tag:

    • It is placed before and after an element. One opening and one closing tag.
    There are self-closing aslo.

  • CSS:

    • It stands for Cascading Style Sheet. It is used for styling HTML sheets.
    Here are three ways of applying CSS.

    • Inline CSS
    • Internal CSS
    • External CSS

  • CSS selector:

    • CSS selector are used to apply CSS features to a specified portion of the sheet, when an external CSS is being applied.

    Types of CSS selectors:
    1. Element Selector
    2. Class Selector
    3. ID Selector
    4. Attribute Selector
    5. Universal Selector
